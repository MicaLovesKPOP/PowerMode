# Security Policy

## Supported versions

Only the latest public release is actively supported.

## Reporting a vulnerability

Please do not post security-sensitive reports publicly.

Open a private contact channel with the maintainer, or create a minimal issue asking for a private contact route without including exploit details.

## Scope

Power Mode modifies Windows power plans and user-level startup behavior. Please include:

- Power Mode version
- Windows version
- installer or portable/test package used
- exact steps to reproduce
- relevant logs if safe to share
